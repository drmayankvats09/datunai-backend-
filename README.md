# Datun AI — Backend

Secure Node.js backend for Datun AI.

## Deploy on Railway

1. Upload this folder to GitHub
2. Connect Railway to GitHub repo
3. Add environment variable: ANTHROPIC_API_KEY
4. Deploy!

## Local Testing

```bash
npm install
cp .env.example .env
# Add your API key in .env
npm start
```
